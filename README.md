# 🔗 Сервис сокращения ссылок

Быстрый и легковесный сервис для сокращения URL, построенный на FastAPI и SQLite.

## 📋 Содержание
- [Особенности](#особенности)
- [Технологии](#технологии)
- [Запуск] (#запуск)

## ✨ Особенности
- ✅ Сокращение длинных URL до 6-символьных кодов
- ✅ Валидация URL (поддержка HTTP, HTTPS, FTP, FTPS)
- ✅ Обнаружение дубликатов ссылок
- ✅ Быстрое перенаправление
- ✅ Простое REST API
- ✅ Легковесная база данных SQLite

## 🛠 Технологии
- **Backend**: FastAPI (Python 3.8+)
- **База данных**: SQLite
- **Алгоритм хэширования**: MD5 + Base62 кодирование
- **Сервер**: Uvicorn

## Запуск

```bash
uvicorn API.short_url_api:app --reload
